Part 1 - Getting Familiar with the Arduino Board

I learned about how each pin functions by using the Arduino board manual and online resources.

Arduino Pins

Digital Pins: Used for basic input/output.

Analog Pins: Can read sensor data.

PWM Pins: Marked with "~", these allow analog-like output.

Power Pins: 3.3V, 5V, GND — used to power external components.

Part 2 – Reading Sensor Data

I practiced 3 simple Arduino sketches using analogRead.

Code 1: Testing Basic Code

Setup: Connected a temperature sensor to A0.


          void setup() {
            Serial.begin(9600);
          }
          
          void loop() {
            int sensorValue = analogRead(A0);
            Serial.println(sensorValue);
            delay(1000);
          }

Result: Serial Monitor printed values like 345, 346, etc., showing analog input.

Code 2: LED Based on Threshold

Setup: Same sensor, LED connected to pin 13.


          void setup() {
            pinMode(13, OUTPUT);
          }
          
          void loop() {
            int value = analogRead(A0);
            if (value > 500) {
              digitalWrite(13, HIGH);
            } else {
              digitalWrite(13, LOW);
            }
          }

Result: LED turns on when sensor value crosses 500.
