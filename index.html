This hands-on project is just a minor part of a big installation. The purpose is to maximize the data accuracy, analyze and find the best way to optimize the functionability of the Arduino Board when it is installed later.

Apparatus Used: Arduino, Adafruit BMP280 Sensor Pins, Adafruit Ultimate GPS Breakout v.3 Pins, Pulse Generator, Oscilloscope Monitor

Challenges:
Coding Accuracy Optimisation

Before I compared how the baud rate and the pulse frequency affected the data accuracy, there was always a problem with the code that decreased the accuracy of the data and significantly increased the memory usage. I looked deeper into how the data was being transmitted, reading the arduino manual and the code tutorial to see how can I modified the code to make a better version of the code that could solve the problems. After reading all of those, I successfully coded a better version that used less memories and increased the data accuracy

Reflections: Since this is my very first experience to apply my theoretical knowledge into a hands-on project (ignoring all labs in class), I know how a real-life project is very different from high school's lab I learned to find potential issues such as the memory usage of a code and the functions of each pin in Arduino, combining all of my knowledge to result in a better version of the code.
