Arduino Project

Part 1 – Getting Familiar with the Arduino Board

I learned about how each pin functions by using the Arduino board manual and online resources.

Arduino Pins

Digital Pins: Used for basic input/output. I tested blinking LEDs using pins 13 and 12.

Analog Pins: Can read sensor data. I practiced using A0 with a temperature sensor.

PWM Pins: Marked with "~", these allow analog-like output. I tried dimming an LED.

Power Pins: 3.3V, 5V, GND — used to power external components.

Part 2 - Reading Sensor Data

Used analogRead to get input from a temperature sensor and display it on Serial Monitor.

Part 3 - Output & Control

Added control logic to light up LEDs based on sensor thresholds.
